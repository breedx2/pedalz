# pedalz

an experimental ux for pdx pedalpalooza

the website is here: [tbd](xxx)

tell your friends, or don't. bikevana don't care!

## setup

```
git clone https://github.com/breedx2/pedalz.git
cd pedalz
npm install
npm run fetch
```

building:

```
npm run build
```

local live-refresh dev server:

```
npm run dev
```
